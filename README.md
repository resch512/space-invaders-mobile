# Space Invaders Mobile

A lightweight mobile-friendly Space Invaders game built with plain HTML5 Canvas + JavaScript.

## Features

- Automatic player shooting
- Touch/drag left-right ship movement
- Invaders move side-to-side, then descend one row on edge hit
- Invaders shoot occasionally
- 3 lives system
- Collision logic: bullet hits and contact damage
- Animated invaders (2-frame walk cycle)
- Procedural sound effects:
  - Alien movement step sound
  - Player laser shot sound

## Run locally

Open `index.html` in a browser.

## Deploy on GitHub Pages

1. Push repo to GitHub
2. Enable Pages for branch `main` and folder `/ (root)`
